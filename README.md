# Lenovo速 Conservation Mode Switcher
KDE Plasma widget to turn on/off the battery conservation mode in **compatible Lenovo速 laptops**.

**If you don't use KDE** you can take a look at the [LenovoConsMode](https://github.com/MaksBal/LenovoConsMode) script.

## Install

### KDE Store
[Lenovo速 Conservation Mode Switcher - KDE Store](https://store.kde.org/p/2074337/)

### Manual
- Download/clone this repo.
- Run from a terminal the command `plasmapkg2 -i [widget folder name]`.

## Disclaimer
I'm not a widget or KDE developer, I did this by looking at other widgets, using AI chatbots, consulting documentation, etc. So use it at your own risk.
Any recommendations and contributions are welcome.
